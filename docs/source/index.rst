Section Pro
===========

**Section Pro** is a Blender addon for creating non-destructive section cuts, generating 2D technical drawings, and organising them into named views. It is designed for architectural visualisation, technical documentation, and any workflow that requires accurate section cuts or 2D projections of 3D models.

- **Version:** 1.0.0
- **Blender:** 4.2 or newer
- **License:** GPL-3.0-or-later
